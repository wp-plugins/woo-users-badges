Let the admin add badges for the users according to a specific rules added from the admin area

== Description ==

= Description =
This plugin allow the store's admin to set rules for the user to get badges, like if the user's orders exceed a specific number ( or decrease ), the user get the badge the admin upload, also if the user registered for sometime, the user can get another specific badge .
= Compare parameters =
1. Equal
1. Not Equal
1. Greater than
1. Greater than or equal
1. Less than
1. Less than or equal
= Shortcode =
1. Add the following shortcode to any page [user_details badge=yes] and for adding the username [user_details badge=yes username=yes] or [user_details badge=yes username=yes]Heading here[/user_details]
== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
